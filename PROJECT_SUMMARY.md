## 🎨 设计系统

### 配色方案
```
主色调: 蓝色-紫色渐变 (#3B82F6 → #9333EA)
功能色:
  - 蓝色 (#3B82F6): 知识、信息
  - 紫色 (#9333EA): 工具、创作
  - 绿色 (#22C55E): 学习、成长
  - 橙色 (#F97316): 连胜、激励
  - 黄色 (#EAB308): 成就、奖励
```

### 组件设计规范
- 圆角: 8px (卡片), 12px (大卡片)
- 阴影: sm (小), md (中), lg (大)
- 间距: 4px 基准，4/8/12/16/24/32/48
- 字体: -apple-system 系统字体栈
- 过渡: 200-300ms ease

## 🎯 核心价值

### 产品定位
这不是一个普通的知识管理工具，而是一个**体验式学习平台**：
- 强调**体验设计**而非信息堆砌
- 关注**产出导向**而非单纯消费
- 构建**社交闭环**而非孤立学习
- 建立**游戏化激励**而非依赖自律

### 差异化优势
1. **3E模型**: 系统化的学习体验设计框架
2. **MLS任务**: 最小学习单元，降低启动成本
3. **72小时闭环**: 快速反馈，缩短认知-行为距离
4. **公共舞台**: 社交压力转化为学习动力
5. **游戏化**: XP、徽章、连胜，持续激励
